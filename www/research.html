Molecule

Molecule is a molecular simulator. With it, the user explores the interaction of bio-molecules in real time, allowing the researcher to investigate biodynamics, protein-protein interaction, gene duplication, etc., in real time and interactively.

Formulae

Formulae most closely resembles a Computer Algebra System (CAS) in the way it is used. The user inputs mathematical information according to Formulae's custom programming language and the interpreter represents the information graphically. However, how Formulae differs from CAS systems is it doesn't represent the information on a Cartesian graph, but as an abstract visual representation. The visual representation is a graphical based language, as opposed to a symbol based language we use in writing and formulating mathematical expressions, equations and theorems. The power of this is not only does this allow the user to visually understand how the different components in a mathematical construct interact, it will also allow the user to input mathematical theorems which will then be translated in to the graphical language. Which, in turn, will allow for the user to see how the different parts of mathematical theorems interact, build on each other, visually.

fysik

fysik is like Formulae. The difference is it's a physics simulator. The user inputs the necessary mathematical machinery to describe the situation he is examining. The mathematical and physical relationships would then be represented in a graphical language the user would then use to explore relationships and connections within the physical system. Additionally, mathematical formula can be modified to explore the consequences if the mathematical relationships were different. For example, if force were equal to the square root of mass times acceleration. Another possibility fysik would be able to explore is the hypothetical scenario where matter is composed of forms of matter different from the standard proton, neutron, electron, etc., and was goverened by different atomic forces.
